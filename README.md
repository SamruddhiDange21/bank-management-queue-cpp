# Bank Management Queue System (C++)

## Description
This is a simple Bank Queue Management System implemented in C++ using a queue data structure.

The system allows the bank to manage customers waiting in line and serve them in FIFO order.

## Features
- Add new customer to queue
- Serve customer
- Display current queue
- Calculate expected waiting time

## Concepts Used
- Queue (FIFO)
- Arrays
- Classes and Objects
- Menu-driven programming
